¿Cómo escribir código en HTML?

Para ello utilizaremos las etiquetas PRE y CODE, la etiqueta PRE es "pre-formateado", mostrará lo que escribiste así mismo, en cambio la etiqueta CODE utiliza un tipo de fuente mono-espaciado para codigo

POr ejemplo: 

    
        #include "librería.h"

        main () {

            //programa principal
        }

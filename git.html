MINI TUTORIAL GIT

  1. Crea un repositorio GIT vacio en el directorio 
                >> git init (directorio)
  2. Clona un repositorio desde un servidor Git. Si no se pasa nombre de directorio se creara con el nombre del repositorio 
                >> git clone https://www.github.com/usuario/miproyecto.git [nombre_directorio]
  3. Indica que archivos se van a preparar para commit 
                >> git add (archivos)
            >> git add .            /* el punto (.) para agregar todos los archivos*/
  4. Realiza un commit con un comentario 
                >> git commit -m "comentario..."
  5. Sube los cambios a una rama de un repositorio 
                >> git push (respositorio) (rama)
  6. Descarga los cambios de una rama de un repositorio y las fusiona con cambios locales 
                >> git pull (repositorio) (rama)
  7. Muestra los archivos modificados en el directorio 
                >> git status
  8. Seguir el log de un repositorio 
                >> git log              /*para mayor detalle*/
            >> git log --oneline    /*menos detalles pero los publica en orden*/

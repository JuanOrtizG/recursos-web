Mini-TUtorial NPM

Una vez que tienes el package.json en la carpeta de nuestro proyecto hacemos, 

    >> npm install

Si queremos instalar un paquete en las dependencias de nuestro proyectos hacemos: 

    >> npm install --save bootstrap

Si queremos instalar un paquete en la dependencias de desarrollo hacemos: 

    >> npm install --save-dev bootstrap
